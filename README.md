# Together Level Manager

这是一个用于管理和处理tgt关卡文件的图形界面应用程序

## 功能特点

- 读取和解析关卡文件
- 以树形视图展示关卡信息
- 支持关卡排序和筛选
- 设置管理功能
- 类型管理功能
- 自动加载和保存设置

## 系统要求

- Python 3.12+
- ttkbootstrap
- Pillow
- colorama

## 安装与运行

### 安装依赖

```bash
pip install ttkbootstrap pillow colorama
```

### 运行程序

```bash
python UI.py
```

## 使用教程

### 1. 初始设置

首次运行程序后，您需要进行一些基本设置：

1. 点击「设置」按钮打开设置窗口
2. 配置以下路径：
   - **压缩包文件夹**：存放关卡压缩包的目录
   - **保存路径**：处理后关卡的保存目录
   - **缓存文件夹**：临时文件存储目录
3. 可选：勾选「自动加载」以在启动时自动加载关卡
4. 点击「保存」按钮保存设置

### 2. 加载关卡

程序会自动扫描配置的压缩包文件夹，解析并显示所有关卡信息。

### 3. 浏览和管理关卡

- 使用树形视图浏览关卡列表
- 点击列标题可对关卡进行排序
- 使用筛选功能查找特定关卡
- 右键点击关卡可进行保存等操作

### 4. 类型管理

点击「类型管理」按钮可以：
- 添加新的关卡类型
- 编辑现有类型
- 删除不需要的类型
